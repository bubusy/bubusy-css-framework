按钮


示例：
default btn-normal btn-success btn-inverse btn-danger default btn-normal btn-success btn-inverse btn-danger

按钮相关的class可以被用于：<a><button><input>标签


示例 class 说明
default .btn 默认样式
btn-normal .btn .btn-normal 普通按钮样式
btn-success .btn .btn-success 绿色系按钮，一般用来表示成功、完成。
btn-danger .btn .btn-danger 红色系按钮，一般用来表示危险按钮。
btn-inverse .btn .btn-inverse 黑底白字按钮

预设了两种按钮大小：正常尺寸、小尺寸。分别对应的class为：.btn.btn-small。

示例： .btn .btn-small
